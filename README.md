# WordPress - PDW 2

Site vitrine pour la société WebAgency en HTML/CSS réalisé pour la 1re soutenance du parcours "Développeur Web Junior" d'OpenClassrooms.

## Contraintes pour le projet
- Il est interdit d’utiliser un thème payant.
- Il n’est pas autorisé d’utiliser un thème dédié aux portfolios, par exemple FSE Freelancer Portfolio.
- L’étudiant doit maximiser l’utilisation de ses compétences de personnalisation de thème.
- Il est autorisé de créer un thème enfant à partir d’un thème standard. Twenty twenty-four, Twenty Twenty-five, Neve FSE, Ollie etc.
- Le portfolio doit être fonctionnel et de qualité professionnelle.
